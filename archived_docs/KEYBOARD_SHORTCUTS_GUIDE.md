# ⌨️ New Keyboard Shortcuts & Features

## Quick Reference

### Keyboard Shortcuts
Press **`?`** anytime to see all shortcuts.

| Shortcut | Action | Use Case |
|----------|--------|----------|
| `Ctrl` + `Enter` | Send message | Quick message send |
| `⌘` + `Enter` | Send message (Mac) | Quick message send |
| `Ctrl` + `K` | Search conversations | Find old chats |
| `Ctrl` + `N` | New conversation | Start fresh |
| `Ctrl` + `Shift` + `S` | Export chat | Save backup |
| `?` | Show this help | View shortcuts |

---

## 💾 Export Your Chats

Click the **📥 Download** button in the top right to export your conversation.

### Available Formats

#### 1. **JSON** 
Best for: Backups, data analysis  
Includes: All metadata, timestamps, traces  
Use: Program analysis, backups

#### 2. **Markdown** 
Best for: Documentation, sharing  
Includes: Formatted text, source URLs  
Use: Share on GitHub, blogs, wikis

#### 3. **HTML** 
Best for: Printing, archiving  
Includes: Styled page, print support  
Use: Archive as webpage, print to PDF

#### 4. **CSV** 
Best for: Excel, data analysis  
Includes: Spreadsheet format  
Use: Import to Excel, Google Sheets

#### 5. **Plain Text** 
Best for: Notes, simple storage  
Includes: Clean text only  
Use: Notes app, simple backup

## 💡 Pro Tips

### Speed Up Your Workflow
```
Ctrl+N     Start new conversation
Type...    Your message
Ctrl+Enter Send instantly (no mouse!)
Ctrl+K     Search old chats
Ctrl+Shift+S Export when done
```

### Save Everything
Before closing browser:
1. Click **📥** button
2. Choose **JSON** format (keeps everything)
3. Save file to backup folder
4. You've got a permanent backup! ✅

### Share a Conversation
1. Click **📥** button
2. Choose **Markdown** format
3. Share the file with someone
4. They can read it easily!

### Analyze Data
1. Click **📥** button
2. Choose **CSV** format
3. Open in Excel/Sheets
4. Create charts & graphs!
